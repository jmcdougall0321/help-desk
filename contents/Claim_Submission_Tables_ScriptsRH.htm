ECM70_INSURANCE_TPA Location Input Form > Claim Submission Details Mini Tab>Insurance TPA

  • Table to control which TPA(s) are active to appear on Location Input Form > Claim Submission Details Mini Tab>Insurance TPA dropdown field.
  • [ECM70_TPA_TYPE_N]
    • Types:
      • 1=WC
      • 2=Auto
        • ESIS
        • Zurich
        • GB
        • CCMSI
        • Travelers
        • Sedgwick
      • 3=Fire
        • ESIS
      • 4=Other Incident
        • ESIS
      • 5=Property Damage
        • ESIS
      • 6=Spill
        • ESIS
      • 7=GL
        • GB
  • Script Sample:

    USE [BRACETRAINING]

    GO

     

    UPDATE [dbo].[ECM70_INSURANCE_TPA]

    SET [ECM70_SYS_ACTIVE_F] = 'N'

    ,[ECM70_SYS_UPDT_USR] = -432

    ,[ECM70_SYS_UPDT_DTM] = GETDATE()

    WHERE ECM70_INSURANCE_TPA_ID = 5

     

    UPDATE [dbo].[ECM70_INSURANCE_TPA]

    SET [ECM70_SYS_ACTIVE_F] = 'Y'

    ,[ECM70_SYS_UPDT_USR] = -432

    ,[ECM70_SYS_UPDT_DTM] = GETDATE()

    WHERE ECM70_INSURANCE_TPA_ID = 32

    GO

ECM115_TPA_CODE_MAPPING

  • Table to map AIC values to TPA values. (certain TPAs, not all – Some TPAs map in each value table (i.e. Nature of Injury, Body Part))
  • [ECM115_TPA_ID]
    • TPA ID from [ECM70_INSURANCE_TPA] (i.e. 23=Liberty Mutual)
  • [ECM115_DATABASE_TABLE_X]
    • Code database table (i.e. MARITAL_STATUS, GENDER)
  • [ECM115_CODE_ID]
    • AIC ID of value (i.e. Gender 1=Female, 2=Male)
  • [ECM115_CODE_X]
    • TPA code of value (i.e. Gender M=Male, F=Female)
