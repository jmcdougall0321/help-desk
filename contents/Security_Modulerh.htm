Security Incidents Module Item

 

The Security Incident Module is similar to the other Incident Modules where you have the Main Tabs at the top, and the mini tabs at the bottom.

 

The Security Incident Module includes a Security Details Main Tab where you can classify the Type of Security Incident, Suspect Status and Items Taken which are all picklist controlled.

 

The following Main Tabs are currently available for customization. If additional fields are requested to be added, please reach out to the AIC Support Team: aicsupport@appsint.com

    • General Details
    • Employee Details
    • Security Details
    • Corrective Actions

Security Short Form Module Item

 

The Security Incident Module includes a Short Form, similar to the Injury/Illness Module where you can give access to individuals to enter limited amount of information to begin with and then further incident investigation can be held within the Long Form.

 

The Security Short Form is available for customization. If additional fields are requested to be added, please reach out to the AIC Support Team: aicsupport@appsint.com

Security Incidents

 

If you do not have access to the Security Incidents Module/Module Item you will not be able to edit/view the Security Incidents within the All Incidents Module Grid. The Form will populate BLANK since you do not have access to the Incident Details.

 

 

 

Security Type Module Item

The Security Type Module Item is a picklist dropdown that populates on the Security Details Main Tab. Please ensure that your values are ACTIVE to populate and show within the dropdown.

 

This form allows for International Translations if enabled and used for your Database.

 

 

 

Suspect Status Module Item

 

The Suspect Status Module Item is a picklist dropdown that populates on the Security Details Main Tab. Please ensure that your values are ACTIVE to populate and show within the dropdown.

 

This form allows for International Translations if enabled and used for your Database.

 

 

Items Taken Module Item

 

The Suspect Status Module Item is a picklist dropdown that populates on the Security Details Main Tab. Please ensure that your values are ACTIVE to populate and show within the dropdown.

 

This form allows for International Translations if enabled and used for your Database.
