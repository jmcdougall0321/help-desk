Task 28579: POLARIS - Claim Submission Issue

 

 

Polaris has the “SEDGWICK.GeneralDetails.Use.LocationCode” Claim Submission Setting set to “Y”. When this setting is in use, Claim Submission looks at the Location Code that is selected on the incident’s General Details tab. If the user selects and saves a Location Code for that record, they should be able to submit the claim.
