IE 11 Settings

 

Please use the following steps to update your Internet Settings:

 

  1. Log into the application.

     

     

     

  2. Select Compatibility View Settings from the Tools dropdown and click the Add button to Add this Website. Select Close.

     

    IE 11 users will need to set Compatibility View Settings as you have done with previous versions. Having IE 11 you no longer have the option to set ALL sites to use compatibility view.

    Note: You will need to do one site at a time. You should see the “Tools” option in the Toolbar which will allow you to set the compatibility view for the site. If you do not see the “Tools” option, press the “Alt” key on the keyboard and the option should appear.

     

  3. Select Internet Options from the Tools dropdown in the IE browser window.
  • On the General tab under the Browsing history section, select Settings.

 

 

    • Select Every time I visit the webpage on the Temporary Internet Files tab.
    • Click OK.

  • On the General tab under Browsing history, select Delete.

  • Select the Temporary Internet files and website files checkbox only.
  • Click Delete.

 

 

Note: Delete Temporary Internet Files if you have had issues with the system on your computer. (This will NOT remove your cookies or history but will ensure that you load a fresh webpage from our site instead of possibly loading an old stored page.)

  1. Click on the Security tab and make sure the Enabled Protected Mode (requires restarting Internet Explorer) checkbox is checked.

 

  • Click on Trusted sites under the Select a zone to view or change security settings.
  • Click on Sites to the right of Trusted sites.

 

  • Click the Add button to add this website to the zone.
  • Please make sure to check the Require server verification (https:) for all sites in this zone checkbox.
  • Select Close.

 

 

  1. Within the Security tab click on Internet.

     

    • Click on Custom Level. . .

       

       

    • Navigate to the Downloads section.
    • Underneath Font download, select Enable.

       

       

       

    • Continue scrolling down to the Miscellaneous section.
    • Underneath Display mixed content, select Enable.

       

       

       

    • Click OK out to exit out of both screens.

       

  2. On the Advanced tab under the Security section, check the Use TLS 1.2 box.

 

 

 

After changing the IE settings you may need to log out of the application, close your IE session, open a new IE window and log back in for the changes to take effect.

 

Once these changes have been made, you will be able to view the application with its full capabilities. Please contact aicsupport@appsint.com if you encounter any issues.
