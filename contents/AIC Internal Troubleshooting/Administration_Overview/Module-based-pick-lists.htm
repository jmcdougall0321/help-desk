Module Based Pick Lists

People Involved Type

Allows User with access to specify the different types of People Involved (ex: Customer, Law
‎Enforcement) that will populate Pick lists throughout the application.

Add People Involved Type

Select Add button

Fill in required fields (red asterisk mark, highlighted to the right)

Optional Fields: Active checkbox, Language Description

Once finished click Save and Close

Cost Type

Allows Users to add values to the Cost Type Pick list and specify which Modules these Picklist
‎values will populate

Add Cost Type

Select Add button

Fill in required fields (red asterisk mark)

Optional Fields: Active check box, Language Description

Optional Mini Tabs: Detailed Request Description, Files/Updates, Communication, Review/Approval

Once finished click Save and Close

Assign Modules

Select Record

Select desired Cost Type

Select the ‘Assign Modules’ button

Highlight desired Available Incident Modules

Click the right arrow to move them to the Selected Incidents field

Click Save and Close
