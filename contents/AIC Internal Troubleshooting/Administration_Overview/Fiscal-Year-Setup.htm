Deleted Cases Module Item

Allows Users with access to view all Cases that have been deleted and contains the option to
‎reactivate them.

Reactivate Case

Select the desired case by highlighting it with one click

Click the Reactivate Case button

Select the Active checkbox

Click Save and Close
