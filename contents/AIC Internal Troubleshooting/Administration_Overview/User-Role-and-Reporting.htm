Communications Setup

Report Logos

  1. Add
    1. Select Add button
    2. Fill in required fields (red asterisk mark, highlighted above)
    3. Optional Field: Description
    4. Once finished click Save and Close
  2. Edit
    1. Highlight desired Report Logo with one click
    2. Select the Edit button at the top
    3. Edit Report Logo as desired
    4. Select Save and Close
  3. Delete
    1. Highlight desired Report Logo with one click
    2. Select the Delete button at the top
    3. Select the Delete button within the popup report
  4. View
    1. Select desired Report Logo
    2. Click the View button at the top
    3. Popup window will allow you to view selected report
  5. Send to Excel
    1. Click Send To Excel
    2. Report will generate and open via Excel
  6. Search
    1. Type your search value in any of the blank spaces within the header boxes
      1. Ex. LOGO NAME, FILE NAME, DESCRIPTION
    2. Click Search
    3. The list will refresh and display items that match your search characters

Review Process Module Item

Allows Users to Edit specific review processes and choose which employees are responsible to
‎review specific Locations.

  1. Edit Review Process
    1. Highlight desired Review Process with one click
    2. Select Edit Review Process
      1. Add: Allows User to add an additional Employee to be a responsible reviewer for the Location selected
  2. Send to Excel
    1. Click Send To Excel Button
    2. Report will generate and open via Excel
  3. Search
    1. Type your search value in any of the blank spaces within the header boxes ex. ORGANIZATION, LOCATION NAME, LOCATION CODE, ADDRESS 1, CITY, STATE, COUNTRY, ACTIVE
    2. Click Search
    3. The list will refresh and display items that match your search characters

Review Process Roles Module

This Module Item allows Users to create Review Process Roles and assign Users to Roles
‎created.

 

  1. Add
    1. Select Add button
    2. Fill in required field (red asterisk mark, highlighted above)
    3. Optional Fields: Search Criteria, Available Employees, Selected Employees
    4. Once Search for Employees is complete, Employees that match the Search criteria will populate the left field (Available Employees) and they can be moved to the right field by highlighting them and clicking the right arrow
    5. Once finished click Save and Close
  2. Edit
    1. Highlight desired Review Process Role with one click
    2. Select the Edit button at the top
    3. Edit Review Process Role as desired
    4. Select Save and Close
  3. Delete
    1. Highlight desired Review Process Role with one click
    2. Select the Delete button at the top
    3. Select the Delete button within the popup report
  4. Send to Excel
    1. Click Send To Excel Button
    2. Report will generate and open via Excel
  5. Search
    1. Type your search value in any of the blank spaces within the header boxes (ex: REVIEW PROCESS ROLE)
    2. Click Search
    3. The list will refresh and display items that match your search characters
