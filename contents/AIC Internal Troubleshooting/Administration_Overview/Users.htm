Users&People

People Module Item

Allows Users to Add, Edit, Delete, Send to Excel, and Search People within the application.
‎Creating a Person account will automatically create a User account for the Person being
‎added. The User account must then be given a Username and Password for the Person to have
‎access to the site. Creating a Person Account and then creating an additional user account for
‎the same Person would cause a duplicate to be made within the system.

Add

Select Add button

Fill in required fields (red asterisk mark)

Fill in desired optional fields

Once finished click Save and Close

Edit

Highlight desired Person with one click

Select the Edit button at the top

Edit Person as desired

Select Save and Close

Delete

Highlight desired Person with one click

Select the Delete button at the top

Select Delete button within the popup report

 

Search

All of the Person Grid Columns are sortable and searchable by either clicking the
‎Heading to sort by alphabetical order or by typing the keywords into the search box and
‎selecting Search

Users

This Module Item contains all the User accounts within the system. Users’ accounts are
‎automatically created when a new Person account is created. In order for a User to have
‎access to the system a Username, Password, Profile, and user Location must be given to the
‎User.

Add

Select Add button

Fill in required fields (red asterisk mark, highlighted above)

Optional Fields: Work Phone, Roles, Updates, Active Check Box

Once finished click Save and Close

Edit

Highlight desired User with one click

Select the Edit button at the top

Edit User as desired

Select Save and Close

 

Delete

Highlight desired User with one click

Select the Delete button at the top

Select the Delete button within the popup report

Search:Type your search value in any of the blank spaces within the header boxes
‎Ex. FIRST NAME, LAST NAME, USERNAME, PROFILE, EMAIL, EMPLOYEE, SITE USER, USE ACTIVE, LAST LOGIN
‎All Grid Columns can be sorted by ascending or descending order by simply clicking the title

Profile:  Profile is the Module Item where Users with access can define the access that certain Profile
‎levels will have. Each User is assigned a specific Profile level based on their position which can
‎range from View Only to Full Accessibility of the system. Within the Profile Module Item Users
‎with access can update the Profile, Add new Profile Levels, and more.

Add:

This will allow Users with access to create an entirely new Profile Level, the accessibility associated with the Profile will be updated later using the Security button.

Select Add button

Fill in required fields (red asterisk mark, highlighted above)

Optional Fields: Level, Notes, User with this profile will have access to all locations Check Box, Active Check Box

Once finished click Save and Close

Edit

Highlight desired Profile with one click

Select the Edit button at the top

Edit Profile as desired

Select Save and Close

Delete

Highlight desired Profile with one click

Select the Delete button at the top

Select the Delete button within the popup report

Security:

This is where Users with access
‎can update the accessibility of Users that
‎have the selected Profile level or limit their
‎abilities to certain form actions (for example
‎the Delete button).

Select desired Profile

Click the Security button at the top

Within the new window there will be a series of Check Boxes that correspond with Modules, Module Items, Grid Buttons, etc.

By simply checking or unchecking the box, you can determine whether or not the chosen Profile will have access to each item

Copy Profile:

This allows Users to copy a
‎selected Profile Level and create a new
‎Profile Level that has the same specifications
‎as the copied Profile. This helps to save time
‎when creating a Profile that is similar to
‎another because the User will not have to
‎start from scratch.

Select desired Profile

Click the Copy Profile button at the top

Popup window will prompt you to name the copied Profile

Optional Field: Level

Select OK

Refresh the Search Grid and the new Profile will appear
