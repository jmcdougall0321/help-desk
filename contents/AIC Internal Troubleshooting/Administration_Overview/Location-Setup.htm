Location Setup

 

  1. Organization Structure
    1. Allows Users to specify the Structures Organization including the corporate level, Divisions,and specific Locations therein  
  • Add/Edit/Delete
    • Right click on the Division
    • Select desired modification (Add/Edit/Delete)
    • Locations can be reordered by simply dragging and dropping the Location on top of the Location you wish for it to go below

 

  1. Locations Module Item
    1. Allows Users to add and modify Locations within the system.

  • Add
    • Select Add button
    • Fill in required fields (red asterisk mark)
    • Fill in optional fields
    • Fill in optional Mini Tabs
    • Once finished click Save and Close
  • Edit
    • Highlight desired Location with one click
    • Select the Edit button at the top
    • Edit Location as desired
    • Select Save and Close
  • Delete
    • Highlight desired Location with one click
    • Select the Delete button at the top
    • Select the Delete button within the popup report
  • Assign Responsible
    • Highlight desired Location with one click
    • Click Assign Responsible
    • Select the Profile Level you wish to populate the dropdown
    • Select the Person from the dropdown to assign as a Responsible Person fro that specific Location
  • Send to Excel
    • Click Send To Excel button
    • Report will generate and open via Excel
  • Search
    • Type your search value in any of the blank spaces in the header boxes
      • Ex. ORGANIZATION, LOCATION NAME, LOCATION CODE, ADDRESS, CITY, STATE, COUNTRY, ACTIVE, EDI CODE, LOCATION ID
    • Click Search
    • The list will refresh and display items that match your search characters
  1. Location Groups Module Item
    1. Allows Users with access to set up and modify Location Groups within the system.

           

  1. Add
    1. Select Add button
    2. Fill in required fields (red asterisk mark)
    3. Optional Fields: Status, Status Note, “Is this the same person AIC should contact for approval?”, Important Events
    4. Optional Mini Tabs: Detailed Request Description, Files/Updates, Communication, Review/Approval
    5. Once finished click Save and Close
  2. Edit
    1. Highlight desired Location Group with one click
    2. Select the Edit button at the top
    3. Edit the Location Group name/Active checkbox as desired
    4. Select Save and Close
  3. Delete
    1. Highlight desired Location Group with one click
    2. Select the Delete button at the top
    3. Select the Delete button within the popup report

  1. Setup Groups
    1. Select desired Location Group
    2. Click the Setup Group button at the top
    3. Select the Organization
    4. The Available Locations box will populate with all available Locations in the selected Organization
    5. Select which Location Group you wish to add or remove Locations from
    6. Add or remove Locations by highlighting
    7. Select Save and Close
  2. Send to Excel
    1. Click Send To Excel button
    2. Report will generate and open via Excel
  3. Search
    1. Type your search value in any of the blank spaces within the header boxes
      1. Ex. LOCATION GROUP DESCRIPTION, ACTIVE
    2. Click Search
    3. The list will refresh and display items that match your search characters
    4. Location Access by User Module Item
      1. Allows Users with access to view a list of Users and which Locations they have access to.
    5. Send to Excel
      1. Click Send To Excel Button
      2. Report will generate and open via Excel
    6. Search
      1. Type your search value in any of the blank spaces within the header boxes
        1. Ex. CLIENT, CLIENT PHONE, CLIENT CONTACT FIRST NAME, CLIENT CONTACT LAST NAME, ADDRESS 1, CITY, COUNTRY, STATE/PROV, POSTAL/ZIP CODE, ACTIVE, ASIGNED TO ALL LOCATIONS
      2. Click Search
      3. The list will refresh and display items that match your search characters
