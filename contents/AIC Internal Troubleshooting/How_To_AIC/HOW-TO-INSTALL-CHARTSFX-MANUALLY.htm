HOW TO INSTALL CHARTSFX MANUALLY

 

  1. Copy the zip file to the desktop of the PC needing the installation.
  2. Open the zip file to view the files
  3. Copy all the files to C:\temp folder
  4. Open a command prompt as Administrator (it must be Administrator)
  5. Type cd\ and hit enter
  6. Type cd temp and hit enter
  7. Type _installchartfx.bat (yes you need the underscore at the beginning)
  8. Once Complete, Type exit from command prompt
  9. Open IE and login to the application
  10. Go to Report Center
  11. Select Report Builder
  12. Locate and select a report that uses Charts
  13. Select preview button at the top
  14. Select Chart Radio button
  15. Select Generate report
  16. You might be prompted to allow ChartFX select allow option.
  17. Chart now Appears.
