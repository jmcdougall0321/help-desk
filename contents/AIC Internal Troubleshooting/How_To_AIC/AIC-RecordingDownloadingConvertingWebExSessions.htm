WebEx – Recording Tutorials

 

*Note*: Before starting the recording process, you will need to schedule a meeting with a toll call-in number.

 

  1. Log-in to: https://appsinternational.webex.com/mw0306lc/mywebex/default.do?siteurl=appsinternational
  2. Go to the previously scheduled meeting
  3. Click “Start Now”
  4. Select Audio Conference
  5. In the Use Phone dropdown select “I will call in”
  6. Call toll number (650) 429-3300
  7. Enter the Accesss Code and Attendee ID
  8. Select “Share desktop” (go to document or application you wish to share)
  9. Select the dropdown menu from the very top of the screen (This menu is hidden)
  10. Select Recorder
  11. Then click red button to record when you are ready.

WebEx – Downloading Conversion Tool

*Note*: You will only need to download the conversion tool once on your computer. Create a folder on the Tdrive to place the recorded tutorial in.

(.arf files)

  1. Log-in to: appsinternational.webex.com
  2. Select the Support tab on the left-hand side
  3. Select “Downloads”
  4. In the Recorder and Players field, select “Recording and Playback” link.
  5. Scroll down to Windows 2000+ for .ARF
  6. Select Download (9MB) – Once downloaded, it is ready to use!

    (.wmv files)

  7. Log-in to: appsinternational.webex.com
  8. Select the Support tab on the left-hand side
  9. Select “Downloads”
  10. In the Recorder and Players field, select “Recording and Playback” link
  11. Scroll down to Windows 2000+ for .WRF
  12. Select Download (Record+Player, 4MB) - Once downloaded, it is ready to use!

WebEx – Convert File to a Windows Media File (*If already have conversion tool)

  1. Log in to WebEx
  2. Go to “My Recorded Meetings” on the left
  3. Select the meeting and click the arrow to the right of the screen
  4. Select “Download”
  5. Select “Save”
  6. Once file is downloaded and the Network Recording Player plays the file, select “File” from the top of the menu.
  7. Go to “Convert”
  8. Select WMV (Windows Media Format)
  9. The file will be uploade and saved on your local drive.
  10. Place the file in the folder on the T drive that was previously created.
