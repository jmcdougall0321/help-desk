OSHA Log by Location Groups

To generate the OSHA Log by Location Groups, you will first need to set up the Location Group.

  1. Add Location Groups

    Starting from the Administration Module, you select Location Groups.

    1. From the Location Group search grid, Click Add at the top of the screen
    2. A Location Group Input Form Screen will appear, with a text box that allows you to add a new Location Group.
    3. When finished remember to save and close.

  2. Set up Location Groups

    After creating the Location group, search and highlight the new entry.

    1. Select Setup Groups
    2. Right click on the newly created main group to create your sub groups.

  3. Assign Locations to the newly created Location Group
    1. Select the Location group, to highlight
    2. Highlight the location(s) in the list box and use the arrow to move the location into the previously selected group. (The tree will refresh to show the latest changes)

  4. Generating a OSHA Log by Location Group
    1. Go to the Report Center module, and click on the Report Center module item.
    2. Click on Injury tab, scroll down to OSHA log by Location Group
    3. When the OSHA Log by Location Group opens, select the desired Location Group from the Location Group dropdown list.
    4. Highlight the Location Group item from the locations provided.
    5. Double check, the correct year is selected. Print report.
