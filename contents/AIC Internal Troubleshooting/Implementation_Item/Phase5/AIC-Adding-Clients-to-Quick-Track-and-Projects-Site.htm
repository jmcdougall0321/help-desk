Adding Clients to the Quick Track/Projects Site

 

Important Note: Prior to beginning this task you need to confirm that you have access to both Quick Track and projects.appsint.com

 

Quick Track Tasks

 

  1. https://quicktrack.appsint.com – Enter Username and Password
    1. Administration  select Clients – Click on the Create New Client Button.
      ‎

 


‎

That takes you to the Client Input Form:

      1. Client Name - Required
      2. Client URL (Production Site) - Not Required
      3. Client URL (Training Site) – Not Required
      4. Click the Add New Contact Tab – Required
          • First Name
          • Last Name
          • E-mail
          • Phone
      1. The Active box should be checked.
      2. Click Save and Close.

    1. Administration  select Tasks – Click on the Create New Task Button.
      1. Format: [Client Name] – Implementation

Projects Site Tasks

 

  1. https://projects.appsint.com – Enter Username and Password
    ‎
  2. Administration Module  Organization Structure Module Item
    ‎


‎

    1. Right click CORPORATE – AIC.
    2. Select - New Organization
    3. Organization Level Name – Type the word “Client”
    4. Organization – [Client Name]


 

  1. Administration Module  Client/Project Module Item
    1. Select Add
      ‎

 


‎

In the Location Input Form

    1. Select the new Client Name in the Parent Level list box.
    2. Name = [Client Name] Implementation
    3. Establishment Name = [Client Name] Implementation
    4. Complete the remaining required fields.
      1. Enter “N/A” for all unknown required fields


‎


‎

  1. Administration Module  Users Module Item (to add the client main contact)
    1. Select Add
    2. Enter First Name, Last Name and E-mail Address of main client contact
    3. Username and Password – Everything before the “@” in their email address.
    4. Profile – CLIENT + ADD TASK

For the fields below user information:

    1. Locations – ONLY assign the new implementation task added
      1. This user should only be able to see the new implementation that was added.
    2. Roles – Assign the following:
      1. Client Contact
      2. Responsible Person – Projects
        ‎


        ‎

  2. Projects Module  Add/Edit Projects Module Item
    1. Select Add
    2. Select the Project in the drop-down
    3. Title - [Client Name] Implementation
    4. Number is the Quick Track Item number
    5. Classification - Minor
    6. Category - General
    7. Type – Implementation
    8. Client Contact – Main Client Contact
    9. AIC project Lead – Implementation Lead for AIC
    11. Click the OK button.
