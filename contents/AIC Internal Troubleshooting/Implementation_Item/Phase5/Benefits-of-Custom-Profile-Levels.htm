Benefits of Custom Profile Levels

 

 

Overview

 

This document will explain the benefits and importance of setting up custom profile levels for the AIC system. Going forward these can be modified and new profiles added to meet the needs of the Organization.

 

The 3 most Common Profile Types are:

  • ADMINISTRATOR
  • RECORDKEEPER (aka End User)
  • READ ONLY ( aka View Only)

 

By separating out different levels, you are granting access or limiting access based on the selections in the Profile Setup Form. How the leveling system works is that the lower the level #, the higher access ability the user will have. For example, if there are 4 levels set up in the application, the level 4's will have the "lowest" level of access, and will not have the ability to change the security setup for any of the level 3's, 2's, and level 1's access. The level 3's will have access to update the security setup for any of the level 4's because they are at a "lower" level. The level 2's can update the security setup for any of the level 3's & 4's etc.

 

ADMINISTRATOR – Level 1

 

This level is used for the head company Administrators who will be the ultimate decision makers for all actions within the application. For example, these users will be Adding/Editing Locations, Modifying the Organization Structure, Adding/Editing Pick Lists, User Setup, and Security Profiles Setup for Users, as well as Alerts setup. Users at this level (Usually only a select few) will have access to everything in the application, as well as access to all locations.

Administrators have the ability to make updates to any profile with a level that is "higher". EX: Level 2's, Level 3's or higher.

 


‎RECORDKEEPER – Level 2

 

This level is used for the users who will be entering in data on a daily basis. For example, these users will be Adding/Editing Incident Records, Entering Employee Hours, Corrective Actions/Activity Calendar, and Basic Reporting including OSHA reports. Recordkeepers usually only have access to 1 Location (Work Location).

Users in Level 2 have the ability to make updates to any profile with a level that is "higher". EX: Level 3's or higher

 

 

READ ONLY – Level 3

 

This level is used for the users who will only be able to View, Print, and/or Pull Reports from the application. These users will be unable to Add, Edit, or Delete any data within the system.

Users in Level 3 (If there are no other levels "higher") will not have the ability to make any updates to any profile, even their own.
