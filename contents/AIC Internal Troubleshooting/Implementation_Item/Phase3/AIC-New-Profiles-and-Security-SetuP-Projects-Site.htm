Steps to set up New Profiles and adjust Security Setup

 

 

The 3 most Common Profile Types are:

  • ADMINISTRATOR
  • RECORDKEEPER
  • READ ONLY

 

Instructions: (ADMINISTRATOR)

 

  1. Check each module and module item for any unused buttons or search grids and be sure to Uncheck the boxes. [Note: These are recommended tips; there is no “exact” way to complete this task]
    • EX: Administration Module
      • Uncheck:
        1. Feature/Issue Form Form (Unused)
    • EX: Injury and Illness Module
      • Uncheck:
        1. Agent Search Grid (Unused)
        2. Carrier Search Grid (Unused)
        3. Claims Adjuster Search Grid (Unused)
        4. Claims Administrator Search Grid (Unused)
        5. Short Form Search Grid (Unused)
        6. International Short Form Search Grid (Unused)
    • EX: Injury and Illnesses Search Grid
      • Uncheck:
        1. Main Tabs > FROI
        2. Mini Tabs > View SSN Form Action (Commonly Do Not want SSN visible)
        3. Mini Tabs > View Date of Birth Form Action (Commonly Do Not want DOB visible)
    • EX: Tools Module
      • Uncheck:
        1. Claim Emails Sent Search Grid
  2. Search grid columns and grid buttons and Turn off any buttons that Client is not set up or ready for.
    • EX: Injury and Illnesses Search Grid
      • Uncheck:
        1. Grid Columns > Submitted to TPA Column (Uncheck until Claim Submission is set up)
    2. EX: Tools Module
      • Uncheck:
        1. Change Request Logs (Not Set Up)
  1. Once you are finished customizing the ADMINISTRATOR profile specific to the Client, Click Save and Close.
  2. Refresh the Modules and recheck to make sure everything works properly.
