Creating New FTP/SFTP Site

SFTP (Secure File Transfer Protocol) is a secure tool which facilitates data access and data transfer over a Secure Shell (SSH) data stream. In order to get access to the SFTP Site you will need to use a FTP application (such as FileZilla a free FTP solution that can be downloaded from the internet). You may need to involve your IT team as there may be an internal application that could be used for this process; and if not then you may need permission to download an application from the internet.
