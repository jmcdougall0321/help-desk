[CLIENT] – [TPA] - Claim Submission Setup

 

Reference: Claim Submission Tables_Scripts.docx

 

 

Setup claim submission:

  • Turn on TPA – [TPA]
    • Table: ECM70_INSURANCE_TPA
  • Assign TPA on Claims Submission Details mini tab – Insurance TPA field
    • Table: ECM71_ORG_ENTITY_INSURANCE_TPA
  • Claim Types (if needed) – Turn on Location Code and Claim Type fields in Variables and Customize
    • Table: IMM82_CLAIM_TYPE
  • Map WC Location Codes
    • Table: ORG_ENTITY_CODE
  • Map Pick List Values to TPA Values
    • Table: ECM115_TPA_CODE_MAPPING
