Users&People People Module Item Allows Users to Add, Edit, Delete, Send to Excel, and Search People within the application. ‎Creating a Person account will automatically create a User account for the Person being ‎added. The User account must then be given a Username and Password for the Person to have ‎access to the site. Creating a Person Account and then creating an additional user account for ‎the same Person would cause a duplicate to be made within the system. Users’ accounts are ‎automatically created when a new Person account is created. In order for a User to have ‎access to the system a Username, Password, Profile, and user Location must be given to the ‎User. Each User is assigned a specific Profile level based on their position which can ‎range from View Only to Full Accessibility of the system. Within the Profile Module Item Users ‎with access can update the Profile, Add new Profile Levels, and more. Add: This will allow Users with access to create an entirely new Profile Level, the accessibility associated with the Profile will be updated later using the Security button.
